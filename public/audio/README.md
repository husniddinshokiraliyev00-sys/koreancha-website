# Audio Files for Listening Exercises

This directory contains audio files for the listening exercises.

## File Structure

- `1-1.mp3` - "안녕하세요" (Hello)
- `1-2.mp3` - "제 이름은 김민수입니다" (My name is Kim Minsu)
- `1-3.mp3` - "한국어를 공부해요" (I study Korean)
- `2-1.mp3` - "오늘 날씨가 좋네요" (The weather is nice today)
- `2-2.mp3` - "식사 맛있게 드세요" (Enjoy your meal)

## Adding New Audio

1. Record or obtain audio files in MP3 format
2. Name files according to the exercise ID (e.g., "1-1.mp3")
3. Place files in this directory
4. Update the `audioUrl` in the listening exercises data

## Audio Quality Requirements

- Format: MP3
- Sample rate: 44.1 kHz
- Bit rate: 128 kbps or higher
- Duration: 2-5 seconds per phrase
- Clear pronunciation, standard Korean accent
